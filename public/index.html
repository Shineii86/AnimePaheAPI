AnimePahe API

Free RESTful API for anime data scraped from animepahe.ch

32 Endpoints Node.js + Express Zero Database

Base URL

https://animepaheapi.vercel.app/api

Quick Start

// Search anime
curl "https://animepaheapi.vercel.app/api/search?query=naruto"

// Get anime info
curl "https://animepaheapi.vercel.app/api/info/one-piece"

// Get episodes
curl "https://animepaheapi.vercel.app/api/episodes/one-piece"

API Endpoints

GET/api/healthHealth check
GET/api/statsAPI stats
GET/api/Home (latest, trending, popular)
GET/api/search?query=Search anime
GET/api/suggestions?query=Search suggestions
GET/api/info/:slugAnime details
GET/api/episodes/:slugEpisode list
GET/api/watch/:slugWatch page data
GET/api/seriesBrowse series
GET/api/az-listA-Z list
GET/api/az-list/:letterA-Z by letter
GET/api/seasonAll seasons
GET/api/season/:nameSeason anime
GET/api/genresAll genres
GET/api/genres/:nameGenre filter
GET/api/category/:nameCategory filter
GET/api/studio/:nameStudio filter
GET/api/tag/:nameTag filter
GET/api/trendingTrending anime
GET/api/popularPopular anime
GET/api/recentRecent releases
GET/api/bookmarkBookmarked anime
GET/api/page/:numPaginated home

Response Format

{
  "success": true,
  "results": { ... }
}

Deployment

# Clone and install
git clone https://github.com/Shineii86/AnimePaheAPI.git
cd AnimePaheAPI && npm install

# Run locally
npm run dev

# Deploy to Vercel
vercel deploy
